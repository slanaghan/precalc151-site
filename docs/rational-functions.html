    Problem 1.4.2.

    Graph \(r(x)=\frac{x}{x+2}\) on the domain of all real numbers. Be sure to label any asymptotes and intercepts.
    ðŸ”—
    ðŸ”—
    Solution 1.
    The solution to ProblemÂ 1.4.2 is outlined below:
    ðŸ”—
    Solution: Step 1: Determine where the function is undefined. In our example, the function is undefined when the denominator is 0. Therefore, by setting \(x+2=0\text{,}\) we get \(x=-2\text{.}\) Therefore, \(x=-2\) will be a vertical asymptote. Vertical asymptotes are values that the function approaches.
    ðŸ”—
    Step 2: Graph the function in an appropriate window on Desmos. Set up the window for \(x\) to be \(-4 \leq x \leq 4\text{.}\)
    ðŸ”—
    Step 3: Draw your own graph. Refer to FigureÂ 1.4.4, the graph from Desmos. On your own graph, draw a dotted vertical line at \(x=-2\text{,}\) and label the intercept at \((0,0)\text{.}\)
    ðŸ”—
    NOTE: In calculus, you will learn that a function can also have horizontal asymptotes. Our example has a horizontal asymptote \(y=1\text{.}\) However, for our purposes, indicating only the vertical asymptotes is enough.
