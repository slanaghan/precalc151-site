    Definition 2.2.1.

    In a right triangle, for an angle \(\alpha\) (alpha), define \(\sin \alpha = \frac{opposite}{hypotenuse}\text{,}\) \(\cos \alpha=\frac{adjacent}{hypotenuse}\text{,}\) and \(\tan \alpha = \frac{opposite}{adjacent}\) (the acronym to remember these is SOHCAHTOA).
